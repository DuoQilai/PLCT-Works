# 南京 RISC-V Lab 教学视频测试开发实习生（开放 2 名）

## 工作内容

在南京 RISC-V Lab 进行现场测试与教学视频制作。  
甲辰计划成员单位 **南京英麒智能** 负责搭建与维护南京地区的 **RISC-V 漂流基地与实验室**。目前实验室设备类型快速增多，包括多款来自 Milk-V、算能、Sipeed等厂商的 RISC-V 开发板，教学与生态观测工作正在逐步扩展。

本岗位面向希望深入了解 RISC-V 开发生态的同学，结合 **RuyiSDK 工具链** 与 **支持矩阵测试项目** 的实战任务，实习生将在 mentor 指导下：

- 测试 RISC-V 开发板上 RuyiSDK 各个组件的运行情况，从镜像刷写到软件功能、工具链和运行时系统的可用性、稳定性与性能。
- 测试（特定） RISC-V 开发板的操作系统、应用软件、工具链和运行时系统的可用性、稳定性与性能。
- 提交测试报告，编写相关文档。
- 编写视频脚本、参与录制/拍摄教学视频、后期剪辑和封面制作；
- （进阶）参与南京RISC-V Lab 展厅/沙龙的宣传与活动素材制作；

本岗位为甲辰计划联合招聘岗位，实习合同与实习证明单位均为 **南京英麒智能**，工作地点位于 **江苏省南京市江宁区麒麟人工智能产业园 B8 栋**。

---

## 入职要求

- **LV2 能力等级** 或具备“一生一芯”学习经历；
- 能够耐心细致地进行开发板系统重装、环境部署与功能验证，并撰写测试复现报告；
- 对视频制作、技术内容传播与文案创作有兴趣，会使用 OBS、剪映等视频工具；
- 熟悉 Linux 基本命令，可独立完成配置与调试操作；
- 能够熟练使用 git 进行团队代码合作；
- 使用过至少一种 Linux 发行版，能够独立看着官方文档进行配置以自己日常使用需求的程度。
- 具备良好的学习能力与沟通习惯，能按计划推进任务；
- **平均每周在南京英麒坐班 2 天**，按照 mentor 的任务要求进行设备操作与录制。
